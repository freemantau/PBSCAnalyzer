
**PBSC Analyzer** - Allows you to analyze PowerBuilder source code in a convenient way.
It has: 
* Advanced syntax highlighting;
* Object navigation;

_**Note:** It uses extracted PowerBuilder objects (\*.sr\*) for analyzing_
